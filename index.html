Understanding Colors in CSS

This paragraph uses a color name (blue).

This paragraph uses a HEX value (#ff6600).

This paragraph uses an RGB value (rgb(100, 150, 200)).

This div has a border and background color applied using CSS.

Explanation:

Color names are predefined words like “red” or “blue.” HEX values represent colors in hexadecimal code, e.g. #ff6600. RGB values use numbers to show how much red, green, and blue are mixed together, e.g. rgb(100,150,200).
